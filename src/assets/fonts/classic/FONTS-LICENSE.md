# Classic design fonts — sources & licenses

These fonts back the "classic" worship-aid design (a faithful, freely-licensed
emulation of the parish's in-house Book Antiqua / Garamond aid). All are
redistributable under open licenses; they are vendored so the app renders the
same everywhere (including serverless bundles with no system fonts).

| File(s) | Typeface | Emulates | Source | License |
|---|---|---|---|---|
| `Classic-Serif-*.otf` | TeX Gyre Pagella (URW P052) | Palatino / Book Antiqua (body + sub-heading labels) | Artifex `urw-base35-fonts` | AGPL + font exception (GUST Font License / OFL-compatible) |
| `Classic-Display-*.{ttf}` | EB Garamond | Garamond (small-caps section headers, cover title) | Georg Duffner / Fontsource | SIL Open Font License 1.1 |
| `Classic-Script-Italic.otf` | URW Bookman (Light Italic) | Bookman Old Style (cover "If you are new…" greeting) | Artifex `urw-base35-fonts` | AGPL + font exception |

Full license texts:
- SIL OFL 1.1 — https://openfontlicense.org
- URW base35 (GUST/AGPL font exception) — https://github.com/ArtifexSoftware/urw-base35-fonts

No changes were made to the font files; they are redistributed verbatim (renamed
for the app's font-role scheme).
