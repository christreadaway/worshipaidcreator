About This Questionnaire

We’re building a tool to automatically generate the weekly Sunday worship aid—the booklet parishioners hold at Mass. Before we write a line of code, we need to understand exactly how the current document gets made, what decisions are yours to make, and what’s dictated by the Church.

There are no wrong answers. If you’re not sure about something, say so—that’s useful data. If a question doesn’t apply to your role, skip it. Write in the margins if you need more space.

This takes most people 20–30 minutes. It will save months of rebuilding things the wrong way.
